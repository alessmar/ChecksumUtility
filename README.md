# Checksum Utility

A simple checksum utility written in Kotlin that allow to compute hash using several algorithm. It uses JavaFX to build the UI.

### Prerequisites

* [Maven](https://maven.apache.org/) - Dependency Management

### Compiling

Compile the project with:


```
mvn clean package
```

## Running

Run the utility:

```
java -jar target/
```
